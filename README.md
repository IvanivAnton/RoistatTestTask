# RoistatTestTask

Реализация тестового задания по парсингу Access log сайта

## Development install 
`composer install`

## Production install 
`composer install --no-dev`

## Usage
`php parser.php /path/to/access.log`

## Tests
Works only at <span style="color: yellow">Development install</span>
<br>
`./vendor/bin/phpunit tests`
